🏆 ACTIVIDADES

🥇 Encender y apagar un LED con retardo.

Instrucciones

  • Desarrolla el siguiente ejercicio con ayuda de tu docente.

Objetivo de la actividad: Comprender el uso de la función digitalWrite() y delay() para generar una secuencia básica con temporizador.

Código base

void setup() {

  pinMode(13, OUTPUT); // Configuramos el pin 13 como salida

}

void loop() {

  digitalWrite(13, HIGH); // Enciende el LED

  delay(1000);            // Espera 1 segundo (1000 milisegundos)

  digitalWrite(13, LOW);  // Apaga el LED

  delay(1000);            // Espera 1 segundo

}

🥇 Parpadeo del LED usando bucle for

Instrucciones

Con ayuda de tu docente desarrolla la siguiente actividad.

  • Objetivo: Comprender la función repetir acciones con una estructura de bucle for, automatizando la secuencia sin repetir código manualmente.

Código Base

void setup() {

  pinMode(13, OUTPUT);

}

void loop() {

  for (int i = 0; i < 5; i++) {      // Repite 5 veces

    digitalWrite(13, HIGH);         // Enciende el LED

    delay(500);                     // Espera medio segundo

    digitalWrite(13, LOW);          // Apaga el LED

    delay(500);                     // Espera medio segundo

  }

  delay(2000); // Espera 2 segundos antes de repetir la secuencia

}

🎃✨ Reto de clase

Instrucciones

Objetivo de la actividad: Desarrollar una secuencia de leds  funcional mediante el uso de Arduino IDE y Fritzing.

  • Los estudiantes, se organizan en grupos pequeños de (3 a 4 integrantes), donde diseñan en el programa Fritzing un circuito básico compuesto por tres LEDs conectados a la placa Arduino Nano. Cada LED debe encenderse en una secuencia específica.
  • Una vez diseñado el circuito, los estudiantes programan dicha secuencia utilizando Arduino IDE, y comprobar si la programación ejecuta correctamente el comportamiento esperado.

Materiales

  • Computadoras con Fritzing y Arduino IDE instalados.
  • Placa Arduino Nano.

Código base (Arduino IDE)

void setup() {

  pinMode(2, OUTPUT);

  pinMode(3, OUTPUT);

  pinMode(4, OUTPUT);

}

void loop() {

  digitalWrite(2, HIGH);

  delay(500);

  digitalWrite(2, LOW);

  digitalWrite(3, HIGH);

  delay(500);

  digitalWrite(3, LOW);

  digitalWrite(4, HIGH);

  delay(500);

  digitalWrite(4, LOW);

}
